O programa é escrito em C++11.

Descrição de uso:

Compilar o arquivo "main.cpp" configurando o compilador para seguir o padrão C++11 ou superior.

Executar o programa em linha de comando, passando o nome do arquivo que contém a instância de teste como argumento.

O resultado do processamento é mostrado na tela e a solução é salva no arquivo "solucao_<nome_da_instancia>.txt"
