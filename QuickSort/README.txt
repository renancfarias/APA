O código é escrito em C++11 e se encontra no arquivo compilável "quicksort.cpp".

Descrição e uso:

O programa lê do arquivo "data_in.txt" (que deve estar no mesmo diretório do executável)
uma sequência de números inteiros que se deseja ordenar. Cada número deve ocupar uma linha diferente.

A saída ordenada é escrita no arquivo "data_out.txt" (que também deve estar no mesmo
diretório do executável).
